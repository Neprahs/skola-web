Správa webu

Upravujte novinky, texty na stránkach, fotky a pridávajte vlastné sekcie. Zmeny sa ukladajú na server.

Prihlásenie administrátora

Online admin funguje len so serverom. Na Verceli prihlásenie nefunguje — spustite server lokálne (cd server → npm start) a otvorte http://localhost:3000/admin, alebo nasaďte backend na Render (pozri nižšie).

Zmeny sa ukladajú na server.

Pridať nový článok

Nadpis, popis a obsah článku upravujete pre každý jazyk zvlášť. Fotky a dátum sú spoločné.

✎

Obsah článku

Odseky, fotky a kombinácie text + fotka

Pridávajte časti článku a vyberte rozloženie (vľavo/vpravo) aj veľkosť fotky.
